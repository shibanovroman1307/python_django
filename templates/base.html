Реєстр лікарських засобів

Ласкаво просимо до системи обліку лікарських засобів.

Можливості системи:

  • Перегляд списку препаратів
  • Додавання нових препаратів
  • Редагування інформації
  • Адміністрування бази даних

Інструкція по запуску проєкту:

  1. 1. Перевірте наявність Python: python --version
  2. 2. Встановіть віртуальне середовище: python -m venv .venv
  3. 3. Активуйте віртуальне середовище:
    - Windows: .venv\Scripts\activate
    - Linux/Mac: source .venv/bin/activate
  4. 4. Встановіть залежності: pip install -r requirements.txt
  5. 5. Виконайте міграції: python manage.py migrate
  6. 6. Створіть суперкористувача: python manage.py createsuperuser
  7. 7. Запустіть сервер: python manage.py runserver
